# Demo Project

Welcome to the Prometheus Demo workspace.

## Contents

- `README.md` — This file
- `devices/` — Device declarations
- `firmware/` — Firmware images and manifests
- `simulations/` — Simulation configs and results
- `agents/` — Agent configurations
- `models/` — ML model artifacts

## Getting Started

1. Open the **Files** app from the dock to browse this workspace.
2. Use **Devices** to register and monitor connected hardware.
3. Run **Simulation** to test failure modes.
4. Open **Assistant** to get AI-powered help.
